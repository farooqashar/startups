# Startup Knowledge Repository

A comprehensive knowledge base for understanding and building startups. This repository contains curated information about startup fundamentals, best practices, funding mechanisms, team building, product development, and common challenges.

## Quick Start

**New to startups?** Start with [Startup Fundamentals](wiki/startup-fundamentals.md).

**Building a startup?** Check out [The Startup Playbook](wiki/startup-playbook.md).

**Raising capital?** See [Funding & Capital](wiki/funding-capital.md).
